# Machine Learning පැවරුම පිළිබඳ විශ්ලේෂණ වාර්තාව (Assignment Analysis Report)

ඔබගේ Machine Learning පැවරුමට අදාළ ලිපිගොනු විශ්ලේෂණය කිරීමෙන් පසු, එහි අඩංගු ප්‍රධාන කරුණු සහ ඔබ කළ යුතු කාර්යයන් පහත පරිදි සාරාංශ කළ හැක. ඔබ මෙම ව්‍යාපෘතිය ගැන කිසිවක් නොදන්නා බැවින්, මෙය ඔබට මග පෙන්වීමක් වනු ඇත.

---

### 1. පැවරුමේ මූලික අරමුණ
මෙම පැවරුමෙන් බලාපොරොත්තු වන්නේ සැබෑ ලෝකයේ පවතින දත්ත සමුදායක් (Real-world dataset) භාවිතා කර, Machine Learning ඇල්ගොරිතම කිහිපයක් ක්‍රියාත්මක කර ඒවායේ ප්‍රතිඵල සංසන්දනය කිරීමයි.

> [!IMPORTANT]
> **Deep Learning (උදා: Neural Networks) භාවිතා කිරීම මෙහිදී තහනම් වේ.** ඔබ භාවිතා කළ යුත්තේ සාම්ප්‍රදායික Machine Learning ඇල්ගොරිතම පමණි.

---

### 2. ඇල්ගොරිතම තෝරාගැනීම
ඔබට පහත ක්‍රමවේද තුනෙන් එකක් තෝරාගත හැක:

*   **Supervised Learning හෝ Unsupervised Learning:** ඔබගේ කණ්ඩායමේ සාමාජිකයින් 4 දෙනෙකු සිටී නම්, ජනප්‍රිය ඇල්ගොරිතම **4ක්** (සෑම සාමාජිකයෙකුටම එක බැගින්) භාවිතා කළ යුතුය.
*   **Reinforcement Learning:** සාමාජිකයින් 3ක් හෝ 4ක් සිටින කණ්ඩායමක් නම්, අවම වශයෙන් ඇල්ගොරිතම **2ක්** භාවිතා කළ යුතුය.
